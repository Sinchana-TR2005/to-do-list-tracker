# To-Do List Maker

A simple and responsive To-Do List web application built using only HTML. This project is designed for beginners who want to understand the structure of a basic webpage and create a task list interface without using CSS or JavaScript.

---

## Features

* Add task input section
* Display list of tasks
* Simple and clean layout
* Beginner-friendly project
* Built completely using HTML only

---
